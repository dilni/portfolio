AI_Study_Buddy – Intelligent Learning Workspace

A modern, high-performance web platform designed to transform dense academic materials into structured, actionable learning assets using advanced LLMs and local file processing.

1 / 11

Project Overview

AI_Study_Buddy transforms academic materials into summaries, flashcards, and quizzes. It combines multi-model AI orchestration with local file parsing to provide a secure, efficient, and holistic study environment featuring integrated productivity tools like a Pomodoro timer.

1. AI & Logic Engine (The Brains & Processing)

This core component handles data ingestion and orchestrates multiple AI models for tailored educational content.

Multi-Model Orchestration

Leverages OpenRouter to provide access to Google Gemini 2.0 Flash, Meta Llama 3.3, DeepSeek R1, and GPT-4o-mini with automated safety fallback logic.

Edge-Side Text Extraction

Uses PDF.js and Mammoth.js for client-side document parsing, ensuring sensitive materials are processed locally before AI interaction.

Context-Aware Tutoring

Implements a 16,000 character persistent context window, allowing the AI to provide answers strictly based on the user's uploaded materials.

2. Learning Interface Component (The Interaction Hub)

A fluid, responsive dashboard built to minimize cognitive load and maximize focus during study sessions.

Study Workspace

Unified interface with Markdown rendering and syntax highlighting for clear hierarchical structures of AI outputs.

Focus Productivity Suite

Built-in Pomodoro Timer with automated state management and session tracking for disciplined learning.

Interactive Active Recall

Flashcard Deck component with 3D-flip animations, derived directly from AI-parsed study materials.

3. Infrastructure & Export Layer (The Foundation)

The technical backbone ensuring speed, type-safety, and portability of study materials.

Full-Stack Architecture

Built using TanStack Start (React 19) for a cutting-edge, type-safe experience from routing to server functions.

Export & Portability

Custom jsPDF generation engine that formats AI outputs into professional, paginated study guides ready for printing.
