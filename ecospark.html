University Project (2024) Climate Action (SDG 13)

EcoSpark – Climate Action Website

An interactive, highly performant front-end web application raising awareness for global climate preservation initiatives.

Interactive Awareness Platform

Project Overview

Designed and built in alignment with the United Nations Sustainable Development Goal 13, this platform serves as an interactive hub featuring responsive telemetry presentations, climate tracking pages, and resource coordination boards.

1. Responsive Interface & Performance Architecture

A modern frontend layout built to ensure immediate content rendering across any standard web device canvas.

Semantic Frontend Architecture

Developed using valid, semantic HTML5 structures to guarantee high accessibility standards across all viewports.

Fluid Responsive Styling

Employs advanced CSS3 layout methodologies (Flexbox grids, media query breakouts, and fluid typography rules) to provide seamless rendering across mobile, tablet, and desktop viewports.

Interactive View Component Layer

Implements vanilla JavaScript layout layers to run dynamic UI components, client-side pagination systems, and user interaction modals seamlessly without performance lag.

2. Platform Multi-Page Integration

A unified digital system coordinating diverse data-heavy content modules into an 8-page educational experience.

System Layout Unification

Coordinates an extensive 8-page responsive application interface. Features specialized views for historical climate charts, interactive action resource boards, and regional global impact maps.

Team Component Assembly

Managed the integration of separate component stylesheets and code scripts into a unified codebase using strict Git branching workflows.

Technical Architecture Summary

Component Layer Core Tech Stack Primary Functional Purpose
Markup Framework HTML5 Semantic Engine Structuring multi-page view hierarchies, image placements, and informational sections.
Presentation Layer CSS3 Grid / Flexbox Architecture Managing layout styling, responsive breakpoints, transitions, and theme layouts.
Dynamic Scripting JavaScript (ES6+ Layout Logic) Handling DOM manipulation, navbar scroll toggles, interactive maps, and card selectors.
Development Ops Git, GitHub Version Control Coordinating team code contributions, running merge conflict reviews, and staging page releases.
